# トレチャーハンター
## サイト概要
### サイトテーマ
全国各地の美味しいお茶を検索したり、自分の飲んだお茶の情報を投稿できるコミュニティサイト

### テーマを選んだ理由
日本茶が大好きで、全国各地に広がる美味しいお茶を探しては購入して飲んでいるのですが、まだ自分が知らない美味しいお茶を飲んでみたく、他のお茶好きの人がどんなお茶を飲んでいるのかを知ることのできるサービスがあれば便利だと考え、このテーマにしました。

### ターゲットユーザ
*日本茶が好きな人

### 主な利用シーン
飲んでみたいお茶を検索する時、自分が飲んだお茶の情報を投稿する時
